## 🔧 使用方法

### 导入 Chrome
1. 打开 Chrome 浏览器
2. 进入 `chrome://bookmarks`
3. 点击右上角 ⋮ 菜单
4. 选择「导入书签」
5. 选择 `bookmarks_final.html` 文件

### 导入 Firefox
1. 打开 Firefox 浏览器
2. 按 `Ctrl+Shift+O` 打开书签管理器
3. 点击「导入和备份」
4. 选择「从 HTML 导入书签」
5. 选择 `bookmarks_final.html` 文件

### 导入 Edge
1. 打开 Edge 浏览器
2. 进入 `edge://favorites`
3. 点击 ⋮ 菜单
4. 选择「导入收藏夹」
5. 选择「从 HTML 文件导入」
